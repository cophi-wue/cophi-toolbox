cophi_toolbox.utils.segment_fuzzy(paragraphs: Iterable[Iterable[str]], segment_size: int = 1000, tolerance: float = 0.05) → Generator[[list, NoneType], NoneType][source]¶

Segment a string, respecting paragraphs.

Args:

paragraphs: Paragraphs of a document as separated entities. segment_size: The target length of each segment in tokens. tolerance: How much may the actual segment size differ from the segment_size?

If 0 < tolerance < 1, this is interpreted as a fraction of the segment_size, otherwise it is interpreted as an absolute number. If tolerance < 0, paragraphs are never split apart.
