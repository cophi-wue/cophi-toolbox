cophi_toolbox.complexity¶

This module provides measures that assess the linguistic and stylistic complexity of (literary) texts.

Regarding the formulas, \(N\) is the number of tokens, and \(V\) the number of types. \(H\) is the number of types occuring only once (hapax legomena), \(D\) is the number of types occuring twice (dislegomena).
